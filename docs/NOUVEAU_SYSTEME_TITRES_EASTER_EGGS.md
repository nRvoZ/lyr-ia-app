## 🎉 Exemples d'Utilisation

### **Afficher un Titre Coloré en React**

```tsx
import { TitleColor } from '../types';
import '../styles/titleColors.css';

// Exemple 1: Titre simple
<span className={`title-color-${titleColor}`}>

// Exemple 2: Badge de titre
```
